## Input-Output Helper Files

|File Group|File Description|
|----------|----------|
|input_output_helper | Basic data type declaration, read/write registers, extract bytes from 32-bit word and tensor, and data type/interpretation conversion|
|input_output_helper-time_measurements|Measure time in microseconds using clock cycles passed and clock speed|
|input_output_helper-utils|Pack/unpack bytes in a word and convert an integer array into a word array|
|input_output_helper-debug|Print a vector or tensor in int8 Q0.7 format|
